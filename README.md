# Network Health Checker - healthyChecky
# 

This simple Network Health Check Tool(healthyChecky) performs the following checks:

- Identifies active network configuration
- Detects the default gateway
- Tests local network connectivity (gateway ping)
- Tests internet connectivity (external ping)
- Verifies DNS name resolution
- Generates a diagnostic report for troubleshooting

And finally, results are saved automatically to a report file.

## FAQ

#### What is this project about?

This project is a simple file that can be used from terminal using python to check whether the network is healthy by testing IP configuration, gateway reachability, internet access, and DNS resolution, and saves the results to a report.


#### What programming language is used for Algorithm?

The specific language used for this project is Python and its Libraries.
